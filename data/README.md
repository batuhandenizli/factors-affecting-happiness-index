# Data Dictionary

This folder contains the raw dataset used for the analysis.

| Column Name | Description |
| :--- | :--- |
| **User_ID** | Unique identifier for each participant |
| **Daily_Screen_Time(hrs)** | Average hours spent on social media daily |
| **Sleep_Quality(1-10)** | Subjective rating of sleep quality (1=Poor, 10=Excellent) |
| **Stress_Level(1-10)** | Subjective stress level (1=Low, 10=High) |
| **Exercise_Frequency(week)** | Number of days exercise is performed per week |
| **Happiness_Index(1-10)** | Overall happiness score (1=Low, 10=High) |
